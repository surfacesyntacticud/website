In version 2.15 of SUD data, released in November 2024, there are 300 corpora:
 - 5 corpora are maintained in the mSUD format (called **Native mSUD**). For these corpora, both mSUD and SUD versions are available in the tgz archive.
 - 9 corpora are maintained in the SUD format (called **Native SUD**)
 - 281 corpora are automatically converted to SUD from the corresponding UD data (version 2.15)

Hence, the full 2.15 release contains 295 SUD corpora, together with the mSUD version for five of them: `Beja-NSC`, `Chinese-Beginner`, `Chinese-PatentChar`, `Northwest_Gbaya-Autogramm` and `Pesh-ChibErgIS`.
Note that UD 2.15 has 296 corpora but one corpus cannot be released in the SUD version, because of its CC license which contain the **ND** (NoDerivative) flags:
 - [UD_Portuguese-CINTIL](https://github.com/UniversalDependencies/UD_Portuguese-CINTIL) &rarr; `License: CC BY-NC-ND 4.0`

## Download all corpora
Download the full set of 295 SUD and 5 mSUD corpora: [sud-treebanks-v2.15.tgz](https://grew.fr/download/sud-treebanks-v2.15.tgz).

# Conversion from UD

  * 281 corpora of SUD 2.15 are obtained by conversion from UD. The version of the data and tools used:
    * Input data: [version 2.15 of UD corpora](http://hdl.handle.net/11234/1-5787)
    * **Grew** conversion rules system: [tag `v2.15` of the conversion system](https://github.com/surfacesyntacticud/tools/tree/v2.15/converter)
    * Tools: [**grew**](https://gitlab.inria.fr/grew/grew) version 1.16.1, [**grewlib**](https://github.com/grew-nlp/grewlib) version 1.16.5 and [**conll**](https://github.com/grew-nlp/conll) version 1.19.1
